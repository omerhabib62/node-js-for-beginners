<h3>Differences in NodeJS and Vanilla JS</h3>
<ol>
<li> Node runs on server while Vanilla JS runs on client side(browser).</li>
<li> Node has Common core modules</li>
<li> CommonJS modules instead of ES6 modules</li>
</ol>
